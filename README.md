# Learn.lk
is a complete educational platform designed to make online learning and student management more efficient and accessible for Advanced Level (A/L) students.

Key Features:

👩‍🏫 For Teachers: 
 • Create classes, 
 • schedule Zoom sessions, 
 • upload recorded lessons & class notes, 
 • assign and manage student assignments.

👨‍🎓 For Students: 
 • View and join classes,
 • attend live Zoom sessions, 
 • watch recorded lessons, and submit assignments online.

👩‍🏫 Teacher & student dashboards

📊 Real-time database management with MySQL

This project strengthened our skills in HTML, CSS, JavaScript, PHP, MySQL, Bootstrap, and UI/UX design, while also teaching us the importance of teamwork, time management, and problem-solving.
